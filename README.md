# Scoop Bucket for SBOMHub CLI

This repository contains the Scoop manifest for installing [SBOMHub CLI](https://github.com/youichi-uda/sbomhub-cli) on Windows.

## Installation

```powershell
scoop bucket add sbomhub https://github.com/sbomhub/scoop-sbomhub
scoop install sbomhub
```

## Update

```powershell
scoop update sbomhub
```

## Manifest

The manifest is automatically updated when a new release is published on the sbomhub-cli repository.
